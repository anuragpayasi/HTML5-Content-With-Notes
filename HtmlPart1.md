# HTML Fundamentals – Day 1

## 1. Introduction to HTML

HTML (HyperText Markup Language) is a standard markup language used to define the structure of web pages.

HTML:
- Is not a programming language
- Does not perform logic or calculations
- Uses tags and elements to describe content
- Defines what content appears on a webpage

Example:

```html
<h1>Hello World</h1>
<p>This is a paragraph</p>
```

---

## 2. How HTML Works (Client-Side Rendering Process)

HTML follows a client-side rendering architecture.

Working Process:

1. HTML code is written in a text editor
2. The file is saved with a `.html` extension
3. A web browser opens the file
4. The browser parses the HTML document
5. The browser renders the visual output

Key Technical Terms:

- Parse → Browser reads and interprets HTML markup
- Render → Browser visually displays the processed content

---

## 3. HTML Versions (HTML5)

HTML has evolved through multiple versions.

HTML5:
- Latest and current HTML standard
- Used for modern web development
- Supported by all modern browsers

Advantages of HTML5:

- Improved semantic structure
- Native support for audio and video
- Better SEO
- Improved accessibility
- Cross-platform compatibility

---

## 4. Standard Structure of an HTML Document

Basic HTML5 Template:

```html
<!DOCTYPE html>
<html>
<head>
    <title></title>
</head>
<body>

</body>
</html>
```

---

## 5. Explanation of HTML Structure

<!DOCTYPE html>
- Declares the document type
- Informs the browser to use HTML5 standards

<html>
- Root element of the document
- Contains all other elements

<head>
- Stores metadata
- Not visible on webpage

<title>
- Defines page title
- Appears in browser tab

<body>
- Contains visible content
- Text, images, links, forms etc.

---

## 6. Complete Example

```html
<!DOCTYPE html>
<html>
<head>
    <title>My First HTML Page</title>
</head>
<body>
    <h1>Welcome to HTML</h1>
    <p>This page demonstrates basic HTML structure.</p>
</body>
</html>
```

Browser Output:
- Title shows "My First HTML Page"
- Heading and paragraph appear on webpage

---

## 7. Important Rules

- Tags are written inside angle brackets <>
- Most tags have opening and closing parts
- HTML is not case-sensitive (lowercase recommended)
- Visible content must be inside <body>
- Proper nesting is required

---

## 8. Common Beginner Mistakes

- Saving file as .txt instead of .html
- Forgetting closing tags
- Writing content inside <head>
- Not refreshing browser
- Incorrect nesting
- Missing <!DOCTYPE html>
